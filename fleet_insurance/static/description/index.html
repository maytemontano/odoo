Módulo Pólizas de Seguro Vehicular

Este módulo permite administrar pólizas de seguros vehiculares

Caracteristícas

Manejo de tipos de Pólizas; Menú. Flota -> Configuración -> Tipos de Pólizas de Seguros:

● Nombre: Nombre o descripción del tipo de póliza; por ejemplo de Responsabilidad Civil, etc.

● Meses de cobertura: Número de meses de cobertura por defecto cuando se creen nuevas Pólizas de seguro de este tipo; será posible modificarlas a nivel Póliza de Seguro

● Días para notificar: El número de días por defecto en que se enviará correo electrónico a los seguidores de esa Póliza de seguro informando que la Póliza esta por expirar; será posible modificarlas a nivel Póliza de Seguro

Pólizas de Seguro Vehícular. Menú Flota -> Pólizas de Seguro:

● Referencia: Referencia o nombre de la Póliza de Seguro

● Vehículo: Vehículo al que corresponde esta Póliza de Seguro

● Fecha de inicio: Fecha de inicio de cobertura

● Fecha de fin: Fecha de expiración de cobertura

● Días para notificar: Número de días de anticipación a la Fecha de fin en que se deben de enviar notificaciones a todos los seguidores de esta Póliza
