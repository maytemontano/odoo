Email Template para Orden de Venta

Este modulo crea un nuevo email template a ser utilizado en Ordenes de Venta

Configuración

Para incluir liga de la orden, hay que modificar el template ingrese al menu Ajustes -> Interfaz del Usuario -> Vistas, buscar la plantilla llamada "mail_sale_custom_layout"

Descomentar la seccion correspondiente.
